CodeSnippets

Short code snippets for markdown file
message
用来模拟多人自由对话，可在对话末尾显示九宫格图片。
{{< message from="self" accountID="twitter/" timestamp="自定义时间" name="name" >}}    这里是自定义的信息内容。 {{< /message >}}
timeline
包括时间、标题、描述内容、标签。
{{< timeline time="2022-06-14" title="填写内容" tags="标签名称" >}}
去掉了一些多余的元素，布局上作了较多的调整。{{< /timeline >}}
todolist
基于memos的清单功能，计划改造为从hugo获取清单内容。
{{< todolist >}}
twitter
引用来自 twitter 的内容，样式为引用样式。
{{< twitter_simple user 1085870671291310081 >}}
